# Animated Sidebar Navigation Menu

A simple sidebar navigation menu built using **HTML** and **CSS**. The sidebar can be opened and closed using the CSS Checkbox Hack, without any JavaScript. This project demonstrates the use of CSS transitions, positioning, and hover effects to create an interactive user interface.

## Features

- Sliding sidebar navigation
- Hamburger menu toggle
- Smooth open and close animation
- Navigation menu with icons
- Social media icons
- Hover effects on menu items
- Background image
- Clean and organized code

## Built With

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome
